    Types #

    Ilusiones
    Ilusiones # Una ilusión óptica es una imagen que engaña el sistema visual, desde el ojo al cerebro, y lo lleva a percibir la realidad de forma distorsionada. Las ilusiones ópticas pueden suceder de manera natural o ser creadas por efectos visuales específicos. Esta es el resultado del análisis de la información que se recibe del sistema visual. Este puede hacer que se perciba un objeto que no está presente, dando una imagen distorsionada de la realidad.
    Kernels
    Masking # La aplicación de máscaras de convolución desempeña un papel muy importante en el campo del procesamiento de imágenes. Se puede aplicar desenfoque, nitidez, relieve, detección de bordes, etc. Kernels # Un kernel, matriz de convolución o máscara es una pequeña matriz que se utiliza en el procesamiento de imágenes para desenfocar, enfocar, resaltar, detectar bordes, etc. Para ello se utiliza la convolución entre el núcleo y una imagen.
